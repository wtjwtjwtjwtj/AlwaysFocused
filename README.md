# AlwaysFocused

AlwaysFocused forces Discord to treat the app as always focused, keeping GIF animating in when tabbed out.
# PLEASE READ THIS PART!!
This has been made for, and only tested for the BetterDiscord Client. Please don't be expected for this to work on any other clients that arent BetterDiscord.

#Install Instructions
1. Open Betterdiscord (Or install it, if you haven't yet)
2. Open your settings and scroll down to the BetterDiscord section.
3. Click on plugins, and click on the folder icon on the top-left of the window, titled "Open Plugin Folder"
4. Drag and drop the downloaded AlwaysFocused.plugin.js into the directory you have been taken to.
5. BetterDiscord should automatically detect the plugin, and it now should be visible in the plugins tab. If it doesn't recognise it, press Ctrl + R to restart Discord and check again.
6. Enable AlwaysFocused to keep GIFs playing while the Discord window isn't focused. This includes regular GIFs in channels, and Nitro GIF Profile pictures in voice calls.
